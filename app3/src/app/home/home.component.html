Hello All Welcome to our Angular SPA

SPA - a single page applcation is an appliction that has only one page and all the activity will happen in that single page and it is possible to interchange the html content of the page through the javascript code associated with it.

This is a demo appliction that is managing tasks of a user to allow him maintain a list of tasks and their status of completion.

We have used bootstrap to decorate our applciation and also to take advantage of its' responsive web design.
